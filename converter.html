Campaign JSON Converter

Paste raw Omnisend campaign data → get clean JSON ready for data.json

How to use
Copy your campaign data from Omnisend and paste it below. Each campaign block should follow this column order.
1
In Omnisend, open your Campaigns report and select all rows for the campaigns you want to add.
2
Copy (Ctrl+C) and paste into the text area below. Tab-separated or line-by-line both work.
3
Set the Starting ID to continue from your last campaign (e.g. if you have 14, set it to 15).
4
Click Convert — review the output, then Copy JSON and paste the new campaign objects into your data.json campaigns array.
Expected column order per campaign block
Row 1Campaign Name
Row 2Segment / Audience
Row 3Channel (Email / SMS)
Row 4"Sent" (status label)
Row 5Date & time (e.g. Mar 21, 2026 at 8:00 AM)
Row 6Recipients sent (number)
Row 7Open rate % (or N/A for SMS)
Row 8Open count (number)
Row 9Click rate %
Row 10Click count (number)
Row 11Revenue (€)
Row 12Orders (number)
